## 摘要
多模态大型语言模型（Multimodal Large Language Models，MLLMs）在推理能力方面表现出了显著的成果，但仍然存在幻觉问题，即生成的文本与视觉内容相矛盾。本文提出了双锚内省解码（Dual-Anchor Introspective Decoding，DaID），这是一种新颖的对比解码框架，通过挖掘模型内部的感知差异动态校准每个词元的生成。具体而言，DaID识别出用于放大视觉事实信号的聚光层（Spotlight layer）和用于抑制文本惯性的阴影层（Shadow layer）。通过利用视觉注意力分布来引导这种双锚选取过程，我们的方法实现了精确的、针对词元的自适应。多项基准测试和不同的MLLM实验结果表明，DaID在显著减轻幻觉的同时还增强了整体推理能力。

## Abstract
Multimodal Large Language Models (MLLMs) have demonstrated remarkable reasoning capabilities yet continue to suffer from hallucination, where generated text contradicts visual content. In this paper, we introduce Dual-Anchor Introspective Decoding (DaID), a novel contrastive decoding framework that dynamically calibrates each token generation by mining the model's internal perceptual discrepancies. Specifically, DaID identifies a Spotlight layer to amplify visual factual signals and a Shadow layer to suppress textual inertia. By leveraging visual attention distributions to guide this dual-anchor selection process, our method ensures precise, token-specific adaptation. Experimental results across multiple benchmarks and MLLMs demonstrate that DaID significantly mitigates hallucination while enhancing general reasoning capabilities.

## 一、研究问题与背景

- **核心问题**：多模态大语言模型（Multimodal Large Language Models，MLLMs）在视觉推理和文本生成中经常产生“幻觉”（hallucination），即生成内容与视觉输入不符的现象。此问题严重影响模型在高风险场景下（如医疗诊断、金融分析等）的可靠性。

- **研究动机**：现有的对比解码（Contrastive Decoding, CD）方法通常依赖外部扰动或多次推理前向传播，导致噪声和高计算开销。本文旨在从模型内部层级信号入手，寻找“自省”（introspective）机制来实现幻觉检测与缓解。

- **总体目标**：提出一种无需额外训练、单次前向计算即可动态调整文本生成的框架，使得生成结果更忠实于视觉内容，同时保持语言流畅性与多模态推理能力。

## 二、方法论：Dual-Anchor Introspective Decoding（D AID）

### 核心思想
通过分析模型各层视觉感知与语言惯性分布差异，动态选取两个内部对比“锚点”：
- **Spotlight Layer（聚光锚）**：处于中间层，视觉感知能力最强，用于强化视觉信息；
- **Shadow Layer（阴影锚）**：处于浅层，语言先验占主导，用于抑制文本惯性。

这种“聚光—阴影”双锚结构模拟模型的自省过程，在生成每个 token 时同时参考视觉信号的高峰与语言噪声的低谷。

### 关键技术与算法流程
1. **注意力引导的视觉评分（Visual Attention Score, VAS）**  
   根据各层对视觉 token 的注意力权重计算视觉依赖度：  
   \[
   VAS_t(l) = \frac{1}{H} \sum_{h=1}^{H} \sum_{k\in V}A^{l,h}_{t,k}
   \]
   其中 \(A^{l,h}_{t,k}\) 表示第 \(l\) 层第 \(h\) 个注意力头在第 \(t\) 个生成位置对第 \(k\) 个视觉 token 的权重。

2. **动态锚点选取**  
   - Spotlight 锚：视觉依赖度达到最大层  
     \[
     L_{t}^{spot} = \arg\max_{l} VAS_t(l)
     \]
   - Shadow 锚：Spotlight 之前视觉依赖度最小的层  
     \[
     L_{t}^{shad} = \arg\min_{l<L_{t}^{spot}} VAS_t(l)
     \]

3. **双锚对比调整解码**
   最终解码层的 logits 被修正为：  
   \[
   L_{t}^{DAID} = [L_{t}^{final} + \alpha L_{t}^{spot}](1+\beta) - \beta L_{t}^{shad}
   \]
   其中：
   - \(\alpha\)：可视信号强化系数（推荐值 0.8）
   - \(\beta\)：语言惯性抑制系数（推荐值 0.2）

4. **自适应可行性约束**  
   限定仅在语义合理的候选 token 集合 \(V'\) 上执行上述调整，避免破坏语言流畅性。

### 技术特点
- 单次前向传播（无额外计算图）
- 训练无依赖（plug-in 式推理模块）
- 由视觉注意力动态驱动的细粒度 token 级校准

## 六、主要结论与发现

- D AID 在所有幻觉基准上均获得最优结果：
  - **POPE Accuracy/F1**：最高达 85.08% / 85.92%，优于已有方法最多 +3.7%。
  - **CHAIR I/S** 幻觉率显著下降（LLaVA-1.5 中下降至 11.3%/35.9%）。
  - 在 MME 感知测试中获得综合分数 633.68，为最佳综合表现。

- 同时提升一般多模态推理能力（如 GQA、VQA v2、Seed I 等基准均正向提升 0.4–2.1%）。

- 双锚对比策略在单次前向下即可大幅降低幻觉且加强视觉—语言一致性。

## 七、优点与创新亮点

- **内省式设计**：由模型自身的层内视觉注意力指导，无需额外数据。
- **动态与细粒度**：每个 token 自适应选择最优层级锚点，具备灵活性。
- **高效率**：单次前向计算实现幻觉缓解，比外部 CD 方法更快。
- **训练无依赖**：适用于多种 MLLM 架构，具有通用性与可插拔性。
- **消融验证充分**：明确区分正负锚贡献，方法机制清晰。
