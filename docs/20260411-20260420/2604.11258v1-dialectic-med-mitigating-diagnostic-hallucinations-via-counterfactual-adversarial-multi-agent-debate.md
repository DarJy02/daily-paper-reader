## 摘要
多模态大型语言模型（Multimodal Large Language Models，MLLMs）在医疗领域中存在严重的确认偏差，常常为了支持初始的、可能错误的诊断假设而虚构视觉细节。现有的思维链（Chain-of-Thought，CoT）方法缺乏内在的纠正机制，因此容易导致错误传播。为弥补这一缺陷，我们提出了 Dialectic-Med，一种通过对抗性辩证法强化诊断严谨性 的多智能体框架。不同于静态共识模型，Dialectic-Med 组织了三个具备特定角色的智能体之间的动态互动：支持者（proponent）负责提出诊断假设；反对者（opponent）配备了新颖的视觉反证模块，主动检索与支持者观点相矛盾的视觉证据进行挑战；调解者（mediator）则通过加权共识图解决冲突。通过显式建模“反证”的认知过程，我们的框架确保诊断推理严格建立在经过验证的视觉区域基础上。在 MIMIC-CXR-VQA、VQA-RAD 和 PathVQA 上的实证评估表明，Dialectic-Med 不仅达到了当前的最佳性能，还从根本上提升了推理过程的可信度。除了准确性之外，我们的方法显著提高了解释的忠实度，并有效缓解了幻觉现象，为单智能体基线方法设定了新的标准。

## Abstract
Multimodal Large Language Models (MLLMs) in healthcare suffer from severe confirmation bias, often hallucinating visual details to support initial, potentially erroneous diagnostic hypotheses. Existing Chain-of-Thought (CoT) approaches lack intrinsic correction mechanisms, rendering them vulnerable to error propagation. To bridge this gap, we propose Dialectic-Med, a multi-agent framework that enforces diagnostic rigor through adversarial dialectics. Unlike static consensus models, Dialectic-Med orchestrates a dynamic interplay between three role-specialized agents: a proponent that formulates diagnostic hypotheses; an opponent equipped with a novel visual falsification module that actively retrieves contradictory visual evidence to challenge the Proponent; and a mediator that resolves conflicts via a weighted consensus graph. By explicitly modeling the cognitive process of falsification, our framework guarantees that diagnostic reasoning is tightly grounded in verified visual regions. Empirical evaluations on MIMIC-CXR-VQA, VQA-RAD, and PathVQA demonstrate that Dialectic-Med not only achieves state-of-the-art performance but also fundamentally enhances the trustworthiness of the reasoning process. Beyond accuracy, our approach significantly enhances explanation faithfulness and decisively mitigates hallucinations, establishing a new standard over single-agent baselines.

## 二、方法论：Dialectic-Med 框架

### 1. 核心思想
- 提出一个**三智能体辩论系统**，通过“反事实对抗辩证”（Counterfactual Adversarial Dialectics）实现诊断自校正。
- 三种角色：
  - **支持者（Proponent）**：提出初始诊断假设与推理路径。
  - **反对者（Opponent）**：通过视觉反证模块（Visual Falsification Module, VFM）主动检索**与假设矛盾的视觉证据**。
  - **调解者（Mediator）**：综合双方意见，基于**动态加权共识图（Dynamic Consensus Graph）**作出最终诊断。

### 2. 关键技术细节

- **视觉反证模块（VFM）**
  - 输入：医学图像 I 和当前诊断假设 H。
  - 生成“反事实视觉探针” (Q_cf)：即针对假设可能矛盾的视觉特征（如“若为肺炎则应出现模糊阴影”）。
  - 利用医学领域预训练的 PubMedCLIP 模型计算图像块相似度，生成**反证注意力图 M_cf**。高注意力值区域代表与诊断不符的视觉证据。
  - 公式核心：  
    - 相似度计算： `s_i = (q^T v_i) / (||q||·||v_i||√d)`
    - 归一化注意力： `α_i = exp(s_i/τ) / Σ exp(s_j/τ)`，其中 τ 为温度系数。

- **辩证推理循环（Adversarial Dialectic Reasoning Loop）**
  - 支持者提出初始假设 h₀。  
  - 反对者使用 VFM 查找反证、计算攻击强度 S_attack。  
  - 若攻击强度超阈值 θ_thresh 则添加“反证边”，并由调解者生成反馈促使假设修订。  
  - 迭代形成**动态共识图 G_t = (V_t, E_t)**，节点表示假设或证据，边表示支持/反驳关系及其权重。
  - 最终诊断 D* 由所有路径累计可信度 Φ(h) 最大者完成：
    -  Φ(h) = Σ exp( 1/|π| Σ log w_uv ) ，体现经多轮证伪后的可靠性。

- **训练目标**
  - 新引入**反事实对齐损失（Counterfactual Grounding Loss, L_CFG）**，促使模型区分支持与否定性视觉证据，细化视觉定位的可靠性。

### 3. 总体流程摘要
1. 支持者生成初步诊断。  
2. 反对者通过 VFM 执行视觉反证攻击。  
3. 调解者评估辩论强度并引导修订。  
4. 经若干轮迭代，形成共识并生成可信诊断与解释。
